Настройки

1. Облако (обязательно)

Старое облако у вас не открывается (Failed to fetch). Нужно один раз подключить Firebase — бесплатно.

  1. Откройте console.firebase.google.com
  2. Войдите через Google
  3. Нажмите Создать проект → имя например fokus-plus → далее, далее
  4. В меню слева: Build → Realtime Database → Create Database
  5. Выберите регион (можно Europe) → режим Start in test mode → Enable
  6. Вкладка Rules — вставьте правила ниже → Publish
  7. Вкладка Data — сверху скопируйте ссылку вида https://….firebasedatabase.app
  8. Вставьте ссылку в поле ниже → Сохранить адрес облака
{
  "rules": {
    ".read": true,
    ".write": true
  }
}

2. Синхронизация телефон ↔ ПК

Данные всегда сохраняются на устройстве и работают без интернета. После сохранения адреса облака нажмите «Создать новый код» и введите его на втором устройстве.

Синхронизация выключена

3. Онлайн-запись для клиентов

Сначала шаги 1 и 2. Потом скопируйте ссылку и вставьте в VK. Клиенты видят только свободные даты и время.
